# Kavya Agrawal Academic Website

This is a free Jekyll-based academic website designed for GitHub Pages.

## How to use

1. Create a GitHub repository named:
   `kavyaagrawal58.github.io`

2. Upload all files from this folder into that repository.

3. Go to:
   Settings > Pages

4. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root

5. Save.

Your website should appear at:
`https://kavyaagrawal58.github.io`

## How to add a blog post

Create a new Markdown file inside `_posts/` using this format:

`YYYY-MM-DD-title-of-post.md`

Example:

`2026-05-05-urban-groundwater-risk.md`

Add this front matter at the top:

---
layout: post
title: "Your Blog Title"
date: 2026-05-05
categories: [Groundwater, Policy, GIS]
---

Then write your article below.

## How to add CV

Put your CV PDF inside:

`assets/files/Kavya_Agrawal_CV.pdf`

Then the CV page will link to it.
